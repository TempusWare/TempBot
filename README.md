# TempBot Commands
Commands start with **/** and use spaces to separate arguments.

### Games
###### /unscramble
Get a word to unscramble. (All words are based off Marvel-related things)
*Abbreviations:* `unscram` `unscr`
**Subcommands:**
- end
- hint
###### /numberguess
Guess a number from 1-10.
*Abbreviations:* `numg` `numguess`
**Subcommands:**
- end
- <range>
###### /mathsquestion
Test your maths skills by answering a random maths question.
*Abbreviations:* `maths`
**Subcommands:**
- end
- <range>
- addition / add
- subtraction / sub
- multiplication / mul
- division / div
###### /scissorspaperrock
Play a 2-player game of scissors-paper-rock.
*Abbreviations:* `rockpaperscissors` `rps` `spr` `handgame`
**Subcommands:**
- end

### Other things
